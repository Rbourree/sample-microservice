# Sample Microservice


## 1 - Description
This is a sample nodejs microservice.



## 2 - Requirements
- Nodejs
- NPM
- docker
- docker-compose



## 3 - Installation

### Up postgres and redis container
 ```sh
 $ docker-compose up -d
```

### Install package
```sh
$ npm install
```

### Run API
 ```sh
$ npm start
```
